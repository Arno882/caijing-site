# 圖片上架規範

採景網站的高級感主要來自圖片品質。

## 每件商品至少準備

```text
1 張封面照
1 張正面照
1 張側面照
1 張細節照
```

## 建議比例

```text
商品封面：4:5
商品詳情：4:5 或 3:4
首頁手機主圖：9:16
```

## 拍攝規則

```text
背景乾淨
光線一致
避免黃光
避免雜物
作品不要歪斜
不要使用 LINE 壓縮圖
```

## 上架前流程

```text
朋友提供原圖
↓
交給你修圖
↓
輸出壓縮後圖片
↓
上傳到後台
↓
手機預覽
↓
公開
```

## 圖片檔案建議

```text
封面寬度：至少 1200px
縮圖寬度：800px 以上
格式：WebP / JPG / PNG
檔案大小：盡量小於 500KB
```
