# 採景後台操作說明｜人性化版本

後台網址：

```text
/admin
```

## 重要說明

目前是 0 元靜態版。

後台不使用 R2、不使用 D1、不使用資料庫。

因此後台資料會先存在目前瀏覽器，適合整理商品草稿與預覽。正式上線仍需要網站管理員更新：

```text
data/sampleProducts.ts
```

## 後台分成三區

```text
商品管理
首頁設定
匯出資料
```

## 新增商品

1. 開啟 `/admin`
2. 進入「商品管理」
3. 填商品名稱
4. 選分類
5. 填價格
6. 選狀態
7. 填封面圖片路徑，例如 `/products/moss-001.jpg`
8. 填簡短介紹、商品介紹、照顧方式
9. 按「儲存商品草稿」

## 圖片怎麼放

圖片先放：

```text
public/products
```

例如：

```text
public/products/moss-001.jpg
```

後台填：

```text
/products/moss-001.jpg
```

## 商品狀態

```text
可詢問：可以私訊購買
已售出：已經賣掉
可客製：可討論類似款
僅展示：只展示
隱藏：不顯示
```

## 匯出資料

後台整理好商品後，進入「匯出資料」：

```text
複製資料
或
下載 JSON
```

再由網站管理員更新正式檔案。
